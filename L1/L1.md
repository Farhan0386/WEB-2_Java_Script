## 1. What is DOM?

**DOM** stands for **Document Object Model**.

The DOM is a **tree-like structure** created by the browser from your HTML code.  
It represents **everything on the webpage** (text, buttons, images, links, etc.) as **objects**.

JavaScript uses the DOM to:

- Read content from the page
- Change content
- React to user actions

ğŸ“Œ **Important:**  
HTML creates the page structure, but **DOM makes the page interactive**.

---

## 2. Why Do We Need DOM?

HTML alone is **static**.

Once a page loads:

- HTML cannot change itself
- HTML cannot respond to clicks or typing

The DOM allows JavaScript to:

- Change text and styles
- Detect button clicks
- Read user input
- Add or remove elements
- Create dynamic web pages without reloading

